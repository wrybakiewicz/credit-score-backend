# credit-score-backend

### Requirements:
- NodeJs
- Makefile
- curl

### How to run:
- get from admin & set `COVALENTHQ_API_KEY` (covalenthq api), `THE_GRAPH_KEY` (the graph api), `BEARER_TOKEN` (twitter api) in `.env` file 
- `make install start` - to start server
- `make request` - in another tab
